"# gheras" 
1. الخصائص المطلوبة في داش بورد الأدمن:
المتجر (Store):
إدارة المنتجات: إضافة/تعديل/حذف المنتجات، تصنيف المنتجات، التحكم في المخزون.
إدارة الطلبات: تتبع الطلبات، تغيير حالة الطلب (تم الشحن، في الانتظار، تم الإلغاء).
إدارة الدفع: تتبع المدفوعات، إدارة طرق الدفع، تأكيد الدفع.
التقارير: عرض تقارير المبيعات، أرباح الشهر، أكثر المنتجات مبيعًا، وتحليل أداء المتجر.
إدارة العملاء: الاطلاع على تفاصيل العملاء، تاريخ الشراء، إدارة حسابات العملاء.
الويكي (Wiki):
إدارة المقالات: إضافة/تعديل/حذف المقالات، تصنيف المقالات.
مراجعة المحتوى: موافقة أو رفض المقالات المدخلة من قبل المستخدمين.
إدارة المستخدمين: السماح للمستخدمين بالتسجيل كـ محررين، والتحكم في صلاحيات الكتابة والمراجعة.
المنتدى (Forum):
إدارة المواضيع والمشاركات: إضافة/تعديل/حذف المواضيع والمشاركات.
إدارة الأعضاء: التحكم في صلاحيات الأعضاء (حظر، تنبيه، رفع رتبة، إلخ).
إدارة الفئات: إضافة/تعديل/حذف الفئات.
التقارير والاحصائيات: عرض عدد المواضيع، التفاعلات، أكثر المواضيع زيارة.
2. الـ APIs المطلوبة:
API للإدارة العامة:
API لإدارة المستخدمين: تسجيل الدخول، تغيير كلمة المرور، إضافة مستخدمين جدد، إدارة الصلاحيات.
POST /api/admin/login
PUT /api/admin/change-password
POST /api/admin/users
API لإدارة المتجر:
API للمنتجات: إضافة، تعديل، حذف، عرض المنتجات.
GET /api/admin/products
POST /api/admin/products
PUT /api/admin/products/{productId}
DELETE /api/admin/products/{productId}
API للطلبات: عرض الطلبات، تحديث حالة الطلب.
GET /api/admin/orders
PUT /api/admin/orders/{orderId}
API لإدارة الدفع: التحقق من المدفوعات.
GET /api/admin/payments
API لإدارة الويكي:
API للمقالات: عرض المقالات، إضافة وتعديل وحذف المقالات.
GET /api/admin/wiki
POST /api/admin/wiki
PUT /api/admin/wiki/{articleId}
DELETE /api/admin/wiki/{articleId}
API للمراجعة: التحقق من المقالات الجديدة التي تحتاج للمراجعة.
GET /api/admin/wiki/pending
API لإدارة المنتدى:
API للمواضيع: عرض المواضيع، إضافة وتعديل وحذف المواضيع.
GET /api/admin/forum
POST /api/admin/forum
PUT /api/admin/forum/{topicId}
DELETE /api/admin/forum/{topicId}
API للمشاركات: عرض المشاركات، حذف المشاركات المخالفة.
GET /api/admin/forum/posts
DELETE /api/admin/forum/posts/{postId}
API لإدارة الأعضاء: حظر الأعضاء، رفع رتبة الأعضاء.
PUT /api/admin/forum/users/{userId}/ban
PUT /api/admin/forum/users/{userId}/promote
3. الجزء الذي سيخرج من الـ Controller الخاص بالـ داش بورد:
الـ Controller الخاص بالـ داش بورد عادةً سيكون مسؤول عن:
إرجاع البيانات: عند طلب البيانات من الـ APIs التي ذكرتها (المنتجات، الطلبات، المقالات، المواضيع).
تنفيذ العمليات: عند الضغط على أي زر لإضافة أو تعديل أو حذف عنصر، مثل منتج أو مقال أو موضوع.
التحقق من الصلاحيات: التأكد من أن المستخدم الذي يحاول إجراء العمليات لديه الصلاحيات اللازمة.
الرد على الطلبات: بشكل JSON أو HTML حسب الحاجة، مثل النجاح أو الفشل في تنفيذ العمليات.
